Le fichier creation.sql permet de creér les differentes tables de la base de donnée.

Le fichier remplissage.sql permet de remplir la base généree precedement avec des données, pour la plupart generées aléatoirement.

Le fichier requetes.sql contient les six requetes demandées dans le sujet ainsi que quelques autre requetes jugée pertinente lors de l'usage de cette base de donnée.
